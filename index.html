PrettyBow is a library of Scala for Statistical Language Modeling, Text Retrieval and Classification.
It is a fast, feature abundant and highly configurable tookit built on the top of Lucene and Weka.
The idea of PrettyBow is inspired by Bow (similar library written in C) and TagHelper (similar library written in Java)

The library provides facilities for:

  • Tokenizing a text file, according to several different methods.
  • Including N-grams among the tokens.
  • Word stemming or lemmatization
  • Supporting many filters such as stop words filter,numeric filter and others.
  • Setting word vector weights according to Binary, TF, TFIDF.
  • Smoothing word probabilities according to Laplace (Dirichlet uniform).
  • Pruning vocabulary(feature selection) by word frequency or information gain or SVM-weight.
  • Building incremental bag of word(BOW)/vector space model(VSM) for big data.
  • Writing all prepared data/feature to disk in a compact format
  • Supporting two data formats: ARFF for weka and generic BOW/VSM feature table.
  • Reading the data/feature from disk in an efficient, sparse fashion.
  • Training by Naive Bayes, SVM, DecisionTree on top of pre-prepared BOW/VSM through Weka
  • Performing test/train splits , cross valuation and confusion matrix.

Usage

Example: create a BOW/VSM using

1, Setting word vector weights by TFIDF

2, minimum word frequency > 5

3, minimum of n-gram = 1

4, maxnium of n-gram = 3

5, enable lemmatization

6, disable stemming

7, removing normal english stop words

BOW.config("TFIDF", 5, 1, 3, true, false, true) // the order of parameters correspond to above sequence
BOW.addIndex(id, text, false) //  add id of document and content of document  
BOW.commitIndex  // index all documents which we added above
BuildArff.buildSparseArff("path to save BOW as training data")  // build BOW/VSM and write to disk by ARFF format 
val trainer = new TrainingModel("path of training file", "path to save model")
trainer.currentModel = "SVM"  // training by SVM
val model = trainer.training  // performing training process
trainer.saveModel(model)      // save the model to disk
